CanvasPrep webversie

Deze map is klaar om te uploaden naar een simpele webhost.

Makkelijkste manier:
1. Pak de zip uit
2. Upload ALLE bestanden samen naar een host zoals tiiny.host
3. Open daarna de link op je telefoon in Chrome
4. Tik op de drie puntjes
5. Kies 'Installeren' of 'Toevoegen aan startscherm' als die optie verschijnt

Belangrijk:
- Houd alle bestanden in dezelfde map
- Open index.html via de webhost, niet als lokaal bestand
